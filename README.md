# 📱SocialApp
## Sobre o projeto
Esse projeto faz parte de uma série em que recupero designs antigos, melhoro eles e os desenvolvo com ferramentas atuais, buscando aprimorar e demostrar minhas habilidades.
Se trata da interface de uma rede social, algo como um instagram ou até mesmo um facebook, com animações fluidas e foco 100% em mobile-first, ou seja, para celular e smartphones.
Aproveitem o projeto e tirem suas prórias conclusões!
Também estarei disponibilizando o Figma do projeto!
